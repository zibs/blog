Sinatra is a ruby gem and DSL for creating web applications in Ruby. It can be thought of as a thin layer that's built on top of Rack as it uses rack to take HTTP requests, parse it, and return the response.

Using Sinatra encourages following the REST architectural style by defining routes using an HTTP method coupled with the matching URL. Sinatra also supports dynamic URLs using variables.

Sinatra also allows us to erb, which is embedded ruby, a special templating system included with Sinatra. The method erb here simply is a method that takes a symbol, which references a file name and returns a string which contains our html. It can also be passed a hash specifying which layout partial to use.
