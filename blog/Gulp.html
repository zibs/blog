Gulp is the "streaming build system". As a build system it take cares of tedious, repetitive tasks such as minification, ulgifying, concatenation, and image optimization, to name a few I've started using already. I'll return to this idea of "streaming" at the end of this post.

Gulp is goiing to make things far easier for me now. Even with a basic Gulp file like 

                    // gulpfile.js

    var gulp = require('gulp');
    var gutil = require('gulp-util');
    var browserSync = require('browser-sync');

    gulp.task('watch', ['browserSync'], function (){
      gulp.watch('app/**/*.html', browserSync.reload);
      gulp.watch('app/css/**/*.css', browserSync.reload);
      gulp.watch('app/js/**/*.js', browserSync.reload);
    })

and the simple command 

                      $ gulp watch

I no longer have to press cmd+r to refresh the page and view any changes, as Gulp watches for modifications to any css/html/js files in my development "app" directory, thanks to node globbing. I save a simple thumb movement and two seconds of time.

A simple tutorial that covers this and other useful features is over at css-tricks.
